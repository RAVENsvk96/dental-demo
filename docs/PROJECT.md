# Zubná ambulancia

> Profesionálny demo projekt vytvorený pre portfólio. Cieľom je vytvoriť modernú webovú stránku na úrovni reálneho klientského projektu.

---

# 1. Prehľad projektu

## Typ podnikania

Moderná súkromná zubná ambulancia.

## Lokalita

Nitra.

## Veľkosť

- 3 zubní lekári
- 2 zdravotné sestry
- 2 ordinácie

## Zameranie

- Preventívna stomatológia
- Dentálna hygiena
- Estetická stomatológia
- Implantológia

---

# 2. Cieľ projektu

Vytvoriť modernú webovú stránku, ktorá:

- buduje dôveru,
- prezentuje služby,
- uľahčuje objednanie termínu,
- reprezentuje vysokú kvalitu ambulancie,
- je vhodná do portfólia.

---

# 3. Cieľová skupina

Primárna:

- vek 25–55 rokov,
- obyvatelia Nitry a okolia,
- ľudia hľadajúci kvalitnú zubnú starostlivosť.

Sekundárna:

- rodičia s deťmi,
- seniori,
- noví obyvatelia mesta.

---

# 4. Primárne CTA

Hlavný cieľ webu:

➡️ Objednať termín.

Sekundárne ciele:

- Zavolať.
- Vyplniť kontaktný formulár.
- Nájsť ambulanciu.

---

# 5. Hodnoty značky

Web má pôsobiť:

- dôveryhodne,
- moderne,
- čisto,
- profesionálne,
- pokojne,
- ľudsky.

Nie luxusne.

Nie extravagantne.

Nie prehnane farebne.

---

# 6. Dizajnové princípy

- Mobile First
- Jednoduchosť
- Výborná čitateľnosť
- Dostatok bieleho priestoru
- Jasná vizuálna hierarchia
- Jedno hlavné CTA
- Konzistentný dizajn

# 9. Starter Kit

Komponenty, ktoré vzniknú:

- Hero
- Feature Grid
- Service Cards
- Team Cards
- Review Cards
- FAQ
- CTA Banner
- Contact Section

Každý komponent musí byť navrhnutý tak, aby bol znovupoužiteľný.

---

# 10. Definition of Done

Každá sekcia musí:

- byť responzívna,
- spĺňať accessibility,
- mať kvalitné TypeScript typy,
- byť pripravená pre SEO,
- byť znovupoužiteľná,
- prejsť npm run build,
- prejsť npm run lint,
- byť pripravená na presun do Starter Kitu.
